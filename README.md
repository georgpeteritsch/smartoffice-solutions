# SmartOffice Solutions

Website für SmartOffice Solutions – Büroservice & Automatisierung für kleine und mittlere Unternehmen.

## Team

- **Katharina Bajo** – Buchhaltung & Steuerliche Themen
- **Georg Peteritsch** – KI & Automatisierung

## Tech Stack

- HTML / CSS / JavaScript (kein Framework)
- Formspree für Kontaktformular
- Hosting: Netlify

## Lokal entwickeln

Einfach `index.html` im Browser öffnen oder einen lokalen Server starten:

```bash
npx serve .
```

## Deployment

Der `main`-Branch wird automatisch auf Netlify deployed, sobald das Repo verbunden ist.

Alternativ: Ordner manuell per Drag & Drop auf Netlify hochladen.
